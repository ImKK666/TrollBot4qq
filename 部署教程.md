# TrollBot 部署教程

本文档将指导你完成 TrollBot 的完整部署流程。

## 📋 部署前准备

### 系统要求
- **操作系统**：Windows 10/11、macOS 10.15+、Ubuntu 18.04+
- **Python版本**：Python 3.8 或更高版本
- **内存**：建议 4GB 以上
- **存储空间**：至少 2GB 可用空间

### 必需服务
1. **OneBot兼容的QQ客户端**（如 go-cqhttp）
2. **OpenAI API 密钥**或兼容的AI服务
3. **QQ机器人账号**

## 🚀 快速部署

### 第一步：环境准备

#### 1.1 安装Python
```bash
# 检查Python版本
python --version
# 或
python3 --version
```

如果没有安装Python，请从 [Python官网](https://www.python.org/) 下载安装。

#### 1.2 克隆项目
```bash
git clone <项目地址>
cd qqbot
```

#### 1.3 创建虚拟环境（推荐）
```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate
```

#### 1.4 安装依赖
```bash
pip install -r requirements.txt
```

### 第二步：OneBot客户端配置

#### 2.1 下载 go-cqhttp
1. 访问 [go-cqhttp发布页](https://github.com/Mrs4s/go-cqhttp/releases)
2. 下载适合你系统的版本
3. 解压到独立目录

#### 2.2 配置 go-cqhttp
```yaml
# config.yml 示例配置
account:
  uin: 你的机器人QQ号
  password: ""  # 建议使用扫码登录

message:
  post-format: string
  ignore-invalid-cqcode: false
  force-fragment: false
  fix-url: false
  proxy-rewrite: ""
  report-self-message: false
  remove-reply-at: false
  extra-reply-data: false
  skip-mime-scan: false

output:
  log-level: warn
  log-aging: 15
  log-force-new: true
  log-colorful: true
  debug: false

default-middlewares: &default
  access-token: ""
  filter: ""
  rate-limit:
    enabled: false
    frequency: 1
    bucket: 1

database:
  leveldb:
    enable: true

servers:
  - ws:
      address: 127.0.0.1:3001
      middlewares:
        <<: *default
  - http:
      address: 127.0.0.1:3000
      timeout: 5
      long-polling:
        enabled: false
        max-queue-size: 2000
      middlewares:
        <<: *default
      post:
        - url: ""
          secret: ""
          max-retries: 3
          retries-interval: 1500
          timeout: 0
```

### 第三步：TrollBot配置

#### 3.1 修改配置文件
编辑 `config.py`，重点配置以下项目：

```python
# OneBot 配置
ONEBOT_WS_URL = "ws://127.0.0.1:3001"  # go-cqhttp WebSocket地址
ONEBOT_HTTP_URL = "http://127.0.0.1:3000"  # go-cqhttp HTTP地址

# 机器人核心配置
TARGET_GROUP_IDS = [123456789]  # 要监听的群号列表
BOT_QQ_ID = 你的机器人QQ号  # 机器人QQ号
BOT_NICKNAME = "你的机器人名称"

# API Keys（必须配置）
OPENAI_API_KEY = "你的OpenAI API密钥"
OPENAI_API_URL = "https://api.openai.com/v1"  # 或其他兼容服务

EMBEDDING_API_KEY = "你的嵌入向量API密钥"
EMBEDDING_API_URL = "https://api.openai.com/v1"  # 或其他兼容服务
```

#### 3.2 创建数据目录
```bash
mkdir -p data/memory_db
```

### 第四步：运行部署

#### 4.1 启动 go-cqhttp
```bash
cd go-cqhttp目录
./go-cqhttp
# Windows: go-cqhttp.exe
```

首次运行会要求登录QQ，按提示完成登录。

#### 4.2 启动 TrollBot
```bash
# 在项目根目录下
python main.py
```

如果需要重置数据库：
```bash
python main.py --reset-db
```

## 🔧 高级配置

### 环境变量配置（推荐）
创建 `.env` 文件来管理敏感配置：

```env
# .env 文件
ONEBOT_WS_URL=ws://127.0.0.1:3001
ONEBOT_HTTP_URL=http://127.0.0.1:3000
OPENAI_API_KEY=你的API密钥
EMBEDDING_API_KEY=你的嵌入向量API密钥
```

然后修改 `config.py` 使用环境变量：
```python
import os
from dotenv import load_dotenv

load_dotenv()

OPENAI_API_KEY = os.getenv("OPENAI_API_KEY")
# ... 其他配置
```

### 性能调优配置

#### 内存使用优化
```python
# config.py 中调整以下参数
DIGESTION_BUFFER_SIZE = 30  # 减少消息缓冲大小
RECENT_MESSAGES_BUFFER_SIZE = 15  # 减少上下文大小
MEMORY_SEARCH_TOP_K = 5  # 减少向量搜索数量
```

#### 响应速度优化
```python
# 使用更快的模型进行快速回复
FAST_LLM_MODEL_NAME = "gpt-3.5-turbo"  # 或其他快速模型
QUICK_REPLY_COOLDOWN_SECONDS = 5  # 减少冷却时间
```

## 🐳 Docker部署（可选）

创建 `Dockerfile`:
```dockerfile
FROM python:3.9-slim

WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt

COPY . .
EXPOSE 8000

CMD ["python", "main.py"]
```

创建 `docker-compose.yml`:
```yaml
version: '3.8'
services:
  trollbot:
    build: .
    ports:
      - "8000:8000"
    volumes:
      - ./data:/app/data
    environment:
      - OPENAI_API_KEY=${OPENAI_API_KEY}
      - EMBEDDING_API_KEY=${EMBEDDING_API_KEY}
    restart: unless-stopped
```

## 📊 监控和维护

### 日志管理
TrollBot会在控制台输出运行日志，建议配置日志文件：

```python
# 在main.py中添加日志配置
import logging

logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s - %(name)s - %(levelname)s - %(message)s',
    handlers=[
        logging.FileHandler('trollbot.log'),
        logging.StreamHandler()
    ]
)
```

部署成功后，机器人将开始监听配置的QQ群组，并根据配置参数进行智能回复和用户分析。
